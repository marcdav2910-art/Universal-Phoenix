## 🌌 Was ist Universaline Star?

**Universaline Star** ist ein vollständiges, kinoqualitatives AI-Betriebssystem-Interface — holografisch, kosmisch und lebendig. Entwickelt für Universal Phoenix als persönliche KI-Begleiterin.

Die Oberfläche fühlt sich an wie ein Fenster in eine lebendige Intelligenz: Sterne ziehen vorbei, der Core atmet, und jede Antwort kommt von einer echten KI — verbunden über n8n.

## ✨ Features

| Feature | Beschreibung |
|---|---|
| 🌍 **Holografischer 3D-Globus** | Three.js-Erdkugel mit echten Natural Earth-Daten. Dreht sich automatisch zu jedem Ort, der im Chat erwähnt wird. |
| 💜 **Echte KI via n8n** | Jede Nachricht geht an einen n8n-Workflow, der Claude als Brain nutzt. Mit lokalem Fallback bei Offline. |
| ⭐ **Star Core** | Atmende holografische Sphäre in der Mitte mit rotierenden Ringen, Kristallen und Sternenergie-Wellen. |
| 🌠 **Kosmischer Hintergrund** | Lebendiges Sternenfeld mit Sternschnuppen, Konstellationen und violettem Nebel. |
| 📊 **Live-Panels** | Stellar Analytics, Friendship Level, Mood, Dream Log, Learning Progress, Memory Constellation, Active Missions. |
| 🎤 **Voice Visualizer** | Waveform mit vier Zuständen: Idle / Listening / Thinking / Speaking. |
| 🔮 **Kristall-Mikrofon** | Pulsierende Aura — Ringe ziehen sich beim Zuhören nach innen, pulsieren beim Sprechen nach außen. |
| 🖥️ **Fenster-System** | Alle Panels sind drag-able, collapsible und resizable. |

## 🚀 Setup

### Lokal öffnen
Einfach `universaline-star.html` herunterladen und im Browser öffnen. Keine Installation nötig.

> Three.js wird über CDN geladen — eine Internetverbindung ist erforderlich.

### Mit n8n verbinden
Das Interface ist bereits mit dem n8n-Backend verbunden. Die URL ist in der Datei konfiguriert:

```javascript
const N8N_URL = "https://universalphnx.app.n8n.cloud/assistant/...";
```

Jede Browser-Session bekommt eine eigene `SESSION_ID`, damit n8n den Gesprächsverlauf im Kontext halten kann.

### GitHub Pages
1. Repo → **Settings** → **Pages**
2. Source: `main` Branch, Ordner `/root`
3. Datei dann erreichbar unter:
   `https://marcdav2910-art.github.io/Universal-Phoenix/universaline-star.html`
